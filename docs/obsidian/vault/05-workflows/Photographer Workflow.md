## Ежедневная работа в CRM

### Утро
1. Dashboard → проверить баннер сегодняшних съёмок
2. Orders → проверить новые заявки
3. Calendar → план на неделю

### В течение дня
- Обновить статусы объектов
- Ответить на заявки (одобрить/отклонить)
- Загрузить готовые фото

### Вечер
- Проверить неоплаченные (Dashboard)
- Отправить квоты клиентам ожидающим подтверждения

## Точки боли (Pain Points)

| Проблема | Влияние | Решение |
|---|---|---|
| Нет real-time обновлений | Пропуск новых заявок | Добавить Firestore listeners |
| Ручная загрузка фото | Долго для 50+ снимков | Firebase Storage + bulk upload |
| prompt() диалоги | Неудобный UX при редактировании | Кастомные модальные окна |
| Нет авто-уведомлений | Клиент не знает статус | Email/WhatsApp триггеры |

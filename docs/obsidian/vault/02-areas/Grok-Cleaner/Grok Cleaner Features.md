## ✅ Реализованные фичи

### Таб 1: Batch Processing (Пакетная обработка)

#### Выбор файлов
- **Статус:** Done
- Выбор входной/выходной папки
- Поддержка 16 форматов (JPEG → RAW)
- RAW → JPEG конвертация через rawpy перед отправкой

#### Определение типа комнаты
- **Статус:** Done
- Авто-определение по имени файла (kitchen/bedroom/living room/bathroom/yard/hallway/balcony)
- Ручное переопределение через выпадающий список
- Каждый тип комнаты = свой промпт

#### 7 режимов освещения
- **Статус:** Done

| Режим | Описание |
|---|---|
| Без освещения | Только AI-очистка, без смены неба |
| ☀️ Дневное небо | Яркое дневное небо, летнее |
| 🌅 Золотой час | Тёплые тона заката |
| 🌄 Закат | Израильский закат, оранжево-розовый |
| 🌆 Сумерки | Вечернее небо, синий час |
| 💡 Яркий интерьер | Усиленное внутреннее освещение |
| 🕯️ Вечерний свет | Тёплый уютный вечер |

#### Прогресс и управление
- **Статус:** Done
- Progress bar (файл N из M)
- Status log (текстовый вывод)
- Кнопка Stop/Cancel (флаг прерывания)
- Auto-retry: 3 попытки при ошибках 500/503
- Fallback: генерация изображения если 404/405

### Таб 2: HDR Bracketing (BETA)

#### Загрузка и группировка
- **Статус:** Done (BETA)
- Загрузка папки или отдельных файлов
- Авто-группировка сетов (по 3 или 5 снимков)
- Ручное управление группами: удалить снимок / удалить группу
- Редактор HDR-промпта

#### Пакетная HDR обработка
- **Статус:** Done (BETA)
- Отправка групп на xAI API
- Суффикс `_HDR` у выходных файлов

---

### Таб 3: Prompts (Промпты)

- **Статус:** Done
- Просмотр и редактирование системных промптов для каждого типа комнаты
- Сохранение в конфиг-файл
- Отдельные промпты по режимам

---

### Таб 4: Settings (Настройки)

- **Статус:** Done
- xAI API ключ (сохраняется в `~/.photodesk/grok_config.json`)
- Папка по умолчанию для вывода

---

### Система промптов

#### 3 базовых режима:

**1. Empty Room Mode (Пустая комната)**
- Удалить всю мебель и личные вещи
- Сохранить архитектурные элементы, пол, стены
- Чистый, пустой результат

**2. Declutter Mode (Деклаттер)**
- Убрать беспорядок, организовать пространство
- Сохранить подлинность интерьера
- ❌ НЕ исправлять физические повреждения
- ❌ НЕ добавлять/убирать мебель

**3. Day-to-Dusk (День в сумерки)**
- Конвертировать дневной свет в израильский закат
- Добавить вечернее освещение окон
- Тёплые тона: оранжевый/розовый/фиолетовый

# PhotoDesk CRM — Обзор системы
*[[01-moc/PhotoDesk MOC|← MOC]]*

---

## Что это

**PhotoDesk CRM** — одностраничное веб-приложение (SPA) на Firebase + GitHub Pages для управления фотобизнесом в сфере недвижимости. Интерфейс на иврите, RTL, тёмная тема.

**Живой URL:** https://edmuntas.github.io/photodesk/photodesk.html
**Firebase проект:** `photodesk-45381`

---

## Для кого

- **Первичный пользователь:** Дорон — владелец Sisa Aero Photography
- **Вторичный пользователь:** клиенты (агенты по недвижимости и офисы) — через публичные страницы

---

## Для чего

| Задача | Решение в CRM |
|---|---|
| Управление съёмками | Listings с 7 статусами |
| Клиентская база | Двухуровневая иерархия: компании → агенты |
| Заказы | Входящие заявки из формы букинга |
| Финансы | Статус оплаты, неоплаченные счета на дашборде |
| Доставка | Статус доставки фото/видео/планов |
| Расписание | Календарь съёмок, доступность |
| Прайс | Пакеты, услуги, генератор кавычек |

---

## Технический профиль

```
Тип:         Single-page application (SPA)
Файл:        photodesk.html (7,315 строк)
Фреймворк:   Vanilla JS + Firebase SDK v10.14.1
CSS:         Custom Properties, Flexbox/Grid, RTL
Шрифт:       Heebo (иврит + латиница)
Auth:        Firebase Authentication (email/password)
DB:          Firebase Firestore
Хостинг:     GitHub Pages
```

## Страницы приложения

| Страница | Иврит | Назначение |
|---|---|---|
| Dashboard | לוח בקרה | KPI, пайплайн, календарь, неоплаченные |
| Listings | נכסים | Список объектов, детальная панель |
| Orders | הזמנות | Входящие заявки из формы |
| Calendar | לוח שנה | Месячный вид, предстоящие съёмки |
| Customers | לקוחות | Компании + агенты |
| Packages | חבילות | Пакеты услуг с ценами |
| Services | שירותים | Основные + дополнительные услуги |
| Availability | זמינות | Рабочее расписание, Шаббат |
| Archive | ארכיון | Мягкое удаление, 90-дневная очистка |
| Settings | הגדרות | Бизнес-данные, логотип |
